# Helen - Frontend

Aplicación de escritorio construida con Tauri (React + Rust).

## Estructura

```
frontend/
├── src/              # Código React
├── src-tauri/        # Backend Rust de Tauri
└── package.json
```

## Desarrollo

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
